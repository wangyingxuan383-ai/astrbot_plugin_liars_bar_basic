# astrbot_plugin_liars_bar_basic

AstrBot QQ 群聊《骗子酒馆基础版》插件（3~5 人）。

## Features
- 群聊开房/加入/开始/质疑/剪线/状态/结束/帮助
- 私聊手牌与多牌出牌：`/酒馆 出 2 4 5`
- 一人一房约束，避免私聊串局
- 出牌超时淘汰、剪线超时自动剪线
- 本地素材（卡牌与炸弹图）

## Files
- `main.py`: 核心插件逻辑
- `assets/cards/*.png`: 卡牌与目标牌图
- `assets/bombs/*.png`: 炸弹状态图
- `_conf_schema.json`: 配置项定义
